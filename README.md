### To run the project, you need to have the following installed:

- Python 3

### Then, you need to install the following packages:

- turtle

### To run the project, run the following command:

- python3 chateau.py

### To move the player, use the following keys:

- &uarr; : move up
- &larr; : turn left
- &rarr; : turn right
- &darr; : move down
